FizzBuzz: A Coding Project

Don't "Buzz" when you need to "Fizz." Thankfully, this program can help with that! Introducing FIZZBUZZ, A javascript coding exercise utilizing loops & logic.

Instructions

Write a program that displays each number from 1 to 100. For each multiple of the "Fizz Number", display "Fizz" instead of the number. For each multiple of the "Buzz Number", display "Buzz" instead of the number. For numbers which are multiples of both, display "FizzBuzz" instead.
